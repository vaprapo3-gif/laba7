Тема: Файли в мові C++

## 🔹 Завдання 1

### Короткий опис задачі
Зчитати 10 цілих чисел з текстового файлу `number.txt` та обчислити середнє арифметичне від’ємних чисел.

### Опис алгоритму розв’язання
1. Відкрити файл `number.txt` для читання.
2. Ініціалізувати змінні для суми та кількості від’ємних чисел.
3. Послідовно зчитувати числа з файлу.
4. Якщо число від’ємне:
   - додати його до суми;
   - збільшити лічильник.
5. Після завершення зчитування:
   - якщо від’ємні числа є — обчислити середнє;
   - якщо немає — вивести повідомлення.
6. Вивести результат на екран.

### Пояснення основних етапів роботи програми
Програма відкриває файл для читання та зчитує числа за допомогою циклу `while`. Умовний оператор перевіряє кожне число на від’ємність. Після обробки всіх чисел обчислюється середнє арифметичне як відношення суми до кількості. Результат виводиться у консоль.

## 🔹 Завдання 2

### Короткий опис задачі
Зчитати матрицю розміром 3×3 з файлу `matrix.txt`, замінити всі від’ємні елементи на 1, вивести результат та записати його у файл `result2D.txt`.

### Опис алгоритму розв’язання
1. Відкрити файл `matrix.txt` для читання.
2. Зчитати елементи матриці у двовимірний масив.
3. Вивести початкову матрицю на екран.
4. Обробити матрицю:
   - якщо елемент від’ємний — замінити його на 1.
5. Вивести змінену матрицю.
6. Відкрити файл `result2D.txt` для запису.
7. Записати результуючу матрицю у файл.

### Пояснення основних етапів роботи програми
Програма використовує файловий потік `ifstream` для зчитування даних та `ofstream` для запису результату. Дані з файлу зчитуються у двовимірний масив за допомогою вкладених циклів. Далі виконується перевірка кожного елемента матриці та заміна від’ємних значень. Після обробки результуюча матриця виводиться у консоль і записується у файл.
